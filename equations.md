The breathing field evolves dynamically according to:

\[
\frac{\partial^2 \phi}{\partial t^2} - \nabla^2 \phi + \frac{dV}{d\phi} = 0

\[
\frac{\partial^2 \phi}{\partial t^2} - \nabla^2 \phi + 4 \lambda \phi (\phi^2 - v^2) = 0
